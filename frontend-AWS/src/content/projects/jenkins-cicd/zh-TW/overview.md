# 專案介紹
> CI/CD

Jenkins CI/CD 是 Docker-oriented pipeline project，用於練習 repeatable build、validation、delivery workflow。

此專案重點是讓 build stages 明確可追蹤，讓 failure 更容易 reproduce，並讓 release handoff 更可控。

# 功能特色
- Jenkins pipeline stages。
- Docker-oriented build flow。
- Source checkout and validation stages。
- Repeatable release preparation。
- Clear pipeline failure boundaries。
