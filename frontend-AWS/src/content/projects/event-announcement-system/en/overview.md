# Project Summary
> Serverless Events

Event Announcement System is a serverless notification platform using AWS API Gateway, Lambda, SNS, and DynamoDB.

It demonstrates event-driven cloud fundamentals: publish events, fan out messages, process them with managed compute, and keep operational logs readable.

# Features
- API-driven event publishing.
- SNS-based fan-out.
- Lambda workers for message processing.
- DynamoDB-backed event metadata.
- CloudWatch-friendly operational flow.
