# Project Summary
> Full-stack AWS

Recipe Sharing App is a full-stack AWS application concept built around React, FastAPI, and DynamoDB.

It demonstrates clear frontend delivery, API-backed operations, and managed data storage for a content-sharing workflow.

# Features
- Recipe listing and detail surfaces.
- API-backed recipe create/read operations.
- DynamoDB access-pattern-first data modeling.
- React frontend with FastAPI backend boundaries.
- AWS-oriented deployment path.
