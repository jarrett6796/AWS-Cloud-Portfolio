# 專案介紹
> Full-stack AWS Recipe Sharing App

Recipe Sharing App 是使用 React、FastAPI、DynamoDB 與 AWS infrastructure services 建立的 full-stack cloud project。

原始 book project 聚焦 basic frontend、backend、database workflow。此實作將專案延伸為更接近 cloud engineering 的架構，加入 custom networking、Application Load Balancer、Auto Scaling Group、CloudFront、IAM roles、CloudFormation Infrastructure as Code。

# 功能特色
- Recipe listing、creation、deletion workflow。
- React frontend connected to FastAPI backend。
- FastAPI REST API for recipe operations。
- DynamoDB-backed recipe persistence。
- AWS networking、load balancing、scaling、delivery layers。
