# 專案介紹
> Linux Hosting

EC2 Apache Website 是 Linux 與 Apache deployment project，聚焦 core cloud infrastructure fundamentals。

此專案展示 VM provisioning、web server setup、inbound traffic control，以及在進入 serverless 或 container abstraction 前需要掌握的基本 operational responsibility。

# 功能特色
- EC2-based Linux hosting。
- Apache web server setup。
- Security group inbound HTTP control。
- Manual infrastructure fundamentals。
- 與 serverless、container hosting 形成清楚對照。
