# Project Summary
> Linux Hosting

EC2 Apache Website is a Linux and Apache deployment project focused on core cloud infrastructure fundamentals.

It demonstrates VM provisioning, web server setup, inbound traffic control, and basic operational responsibility before moving to higher-level serverless or container abstractions.

# Features
- EC2-based Linux hosting.
- Apache web server setup.
- Security group inbound HTTP control.
- Manual infrastructure fundamentals.
- Clear contrast with serverless and container hosting.
