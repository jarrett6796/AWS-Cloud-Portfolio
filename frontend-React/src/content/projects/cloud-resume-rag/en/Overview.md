# Project Brief

**AWS Cloud Resume Challenge + GCP RAG** is a multi-cloud engineering portfolio that combines Amazon Web Services (AWS) and Google Cloud Platform (GCP) to demonstrate modern cloud architecture, serverless application development, and Generative AI integration.

The platform enables recruiters, hiring managers, and other visitors to explore the portfolio through an interactive web interface and an AI-powered assistant. Using a **Retrieval-Augmented Generation (RAG)** , the AI assistant provides answers about the candidate(Project Author), projects, system architecture, and implementation details.
